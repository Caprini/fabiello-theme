header.php

header.php is located in template folder, and it contains the header of the theme.

In the tag "HEAD" We've added almost all scripts and CSS template. the "HTML" tags are also open and "BODY", and the "main container".

Scripts "JSSOR Slider" are within the "BODY" and should not be changed unless you know exactly what you're doing.

We made an "include" navigation menu (navigation.php) that displays the menu is made (see section on the menu)

And we begin the Slider. The slider is made with "jssor slider" to make easier the creation of the same. You can visit the official link and generate your own slider: http://www.jssor.com/demos/full-width-slider.slider

Once the file generated on the website we overwrite each of the parts of the template with the generated code. You must take good pieces and overwrite the correct part for everything to work. Do not worry, it's very intuitive.

If you have problems with the slider you can go to the official support forum: http://forovagos.es directly or you can open a ticket on GitHub: http://github.com/caprini/fabiello-theme

navigation.php

navigation.php is located in template folder, and it contains the menu of the theme.

The menu is not very complex, and can be changed without problem according to the standard of 3 Twitter Bootstrap.

The file is very simple and only consists of 32 lines.

our-story.php

our-story.php is located in template folder, and it contains the "our story" section of the theme.

This is a brief explanation about your story, or your company.

Edit is simple. The default path of the logo that appears is "template / img / Fabiello-logo.png" and you just have to modify to the file we want to implement as logo. The image should be 100 x 63px, or have a relationship with this measure (Double would be 200 x 126px).

We can also modify the text of a story in this file.

video-section.php

video-section.php is located in template folder, and it contains the video section of the theme.

This file contains the section with background video.

Change the icon it is as simple as changing the class of the i tag. You can find a list of icons that you can use and its functions: https://fortawesome.github.io/Font-Awesome/icons/

You can also change the title and description that appear and customize it to your liking and take the link to your video.

In this file video1.js call to make the video background work properly.

If you have questions about this matter you can go to the support forum: http://forovagos.es

expertise.php

expertise.php is located in template folder, and it contains the features of the theme.

In this section we handle the section "features" of the company or person.

The operation is very simple, we only have to change the icons and descriptions at will. We can add links if we direct the visitor to another section of a page, or to another external link.

team.php

team.php is located in template folder, and it contains the team panel of the theme.

In this section we show our team.

The operation is very simple, and just change the image fields, and captions.

All the "core" of this file resides in the style.css located in the CSS directory. There is where we can change the visual options in this section. Remember that this template uses Bootstrap 3 Twitter, so you can play with this framework classes to facilitate your work.

our-work.php

our-work.php is located in template folder, and it contains the portfolio of the theme.

This file contains a small bar with a link to Github. This bar can be modified to taste. The icons used are those of Font Awesome: https://fortawesome.github.io/Font-Awesome/icons/

You can edit the thumbnails in the gallery using images of 400x300px.

Edit images of the pop-up is very simple, it is done by the "modal" boxes underneath. Only we must modify the link to the image, title (is displayed above the pop box) and lel footer (the description shown below).

You'd have your fully configured portfolio section.

Remember that for any questions you can visit the support forum: http://forovagos.es

people-say.php

people-say.php is located in template folder, and it contains the opinion panel of the theme.

In this file we find a carousel where we can put the opinions or recommendations that we have collected.

This section uses blockquotes.js found in the js directory.

contact_form.php

contact_form.php is located in template folder, and it contains the contact form of the theme.

In this file we find the visual part of our contact form.

Here we can change the field names (not the id, but placeholders and the text between the tags). It could change the subject drop down directly from here.

We may modify the side table with our contact.

The form uses the file contact.php that makes the check function and form submission. There modify contact details so that emails reach our mail, and can also edit the message texts sent, or failed attempt (to modify it aspect must use classes bootstrap and the styles defined in the style.css

footer.php

footer.php is located in template folder, and it contains the footer of the theme.

Here are two key elements, the first is the legal text, copyright or that are in the footer, and the other is the up scroll button.

The legal text is very simple to modify. Remember to leave my attribution text in the footer, but you can add whatever you want.

The scroll button is controlled by a script that is directly in the same file. As is the footer this should not hinder the burden of our website.

In the style.css can change the appearance of this object.

style.css

style.css is located in css folder, and it contains the style of the theme.

Any visual change we want to make our template we do it through style.css for proper operation.

The file is well commented, and auqnue still have no relationship classes and id to put in this manul think them soon.
